Exercícios de Java – Fundamentos e POO

Este repositório reúne exercícios desenvolvidos durante meus estudos em Java com foco na consolidação dos fundamentos da linguagem e dos principais conceitos de Programação Orientada a Objetos.

Os exercícios abordam:

Estruturas de dados com List

Manipulação de coleções

Encapsulamento

Criação de classes e métodos

Construtores

Modificadores de acesso

Interação via Scanner

Operações matemáticas e cálculos percentuais

Estruturação de programas em camadas (application e entities)

 Conceitos aplicados

Durante os exercícios foram praticados conceitos como:

Organização de código em pacotes

Separação de responsabilidades

Criação de métodos de negócio (ex: aumento de salário)

Uso de listas dinâmicas (ArrayList)

Busca e filtragem de elementos

Boas práticas de estruturação de projeto Java

Tecnologias utilizadas

Java 17

IntelliJ IDEA

Git e GitHub

Objetivo

Consolidar a base da linguagem Java e evoluir gradualmente para projetos mais estruturados, aplicando boas práticas de desenvolvimento e versionamento de código.
